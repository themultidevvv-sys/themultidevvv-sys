# Proyectos Unity - Elias Taveras

## Sistema Bancario (Proyecto Propio)

**Tecnologías:** C# + SQL Server  
**Descripción:** Sistema bancario completo con conexión remota a Microsoft SQL Server. Implementación de operaciones CRUD, gestión de cuentas y lógica de transacciones.

### Características principales:
- **Gestión de cuentas:** Crear, modificar y eliminar cuentas bancarias
- **Transacciones:** Depósitos, retiros y transferencias entre cuentas
- **Seguridad:** Validación de credenciales y manejo de sesiones
- **Reportes:** Generación de estados de cuenta y movimientos

**Estado:** En desarrollo activo
